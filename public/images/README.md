# Images

Drop your image files in **this folder** (`public/images/`). The app references
them by the exact file names below — **names are case-sensitive on Vercel**, so
keep them spelled and capitalized exactly as shown. If a file is missing, the
card automatically shows a stock photo, so the site never looks broken.

## Notes

- To rename or repoint any image, edit the `image` paths in
  `src/data/articles.js`, the hero in `src/pages/Home.jsx`, or the banner in
  `src/pages/MoodTracker.jsx`.
- Recommended sizes: article images **800 × 500**, hero/banner **~1200 × 600**,
  video thumbnails **800 × 450** px.
